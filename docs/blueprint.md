# **App Name**: Minuto com Deus Pai

## Core Features:

- Mensagem Diária: Exibe uma mensagem inspiradora diferente a cada dia do ano, garantindo que todos os usuários vejam a mesma mensagem diariamente.
- Persistência Local: Utiliza localStorage para armazenar dados de uso, eliminando a necessidade de login e garantindo uma experiência de usuário contínua.
- Compartilhamento Simplificado: Implementa a Web Share API para permitir o compartilhamento fácil da mensagem diária via WhatsApp, promovendo o engajamento viral.
- Carregamento Instantâneo: Gera um arquivo JSON estático com as mensagens para garantir tempos de carregamento mínimos e uma experiência de usuário suave.

## Style Guidelines:

- Cor primária: Azul suave (#A0CFEC) para refletir serenidade e paz.
- Cor de fundo: Branco suave (#FAFAFA) para um design minimalista e limpo.
- Cor de destaque: Verde-azulado (#77DDE7) para botões de compartilhamento e elementos interativos.
- Fonte do corpo e título: 'Literata', uma fonte serifada com um toque vintage e literário.
- Design minimalista com foco em um card centralizado para exibir a mensagem diária, otimizado para visualização mobile-first.
- Animações sutis ao revelar a mensagem diária para adicionar um toque de interatividade sem distrair do conteúdo principal.
- Ícones simples e elegantes para funcionalidades de compartilhamento.
